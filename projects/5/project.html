Portal Teknik Elektro V3

"Portal Teknik Elektro UNS" was developed to address administrative challenges related to student internships and final projects at universities. This system supports students, faculty, and staff by providing resources from project guidelines to final report submissions. Developed using the Rapid Application Development (RAD) method and the Laravel framework with the MVC (Model, View, Controller) architecture, the system allows for fast and structured development, facilitating future enhancements. Successfully piloted in the Electrical Engineering program at UNS, the system is planned for implementation in other programs as well.
