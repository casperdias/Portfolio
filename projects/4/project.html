Design of a Room Visitor Monitoring System and Automatic Door Based on a Website Using ESP32 Cam Microcontroller

This project involves the design of a web-based visitor monitoring system to address issues associated with conventional attendance methods. The system utilizes RFID for data reading, an ESP32 Cam for attendance validation, and a website interface for monitoring. Visitors are required to scan their RFID to enter a room. If the data matches the database, the ESP32 Cam captures an image of the visitorâ€™s face, which is then stored in the database. Admins can monitor the entry and exit of visitors through the website in real-time.
